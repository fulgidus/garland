# skills

Personal AI skills repo. Installable via the [Agent Skills CLI](https://github.com/vercel-labs/skills).

## Install

```bash
npx skills add fulgidus/skills
```

Install a specific skill:

```bash
npx skills add fulgidus/skills --skill coding-best-practices
npx skills add fulgidus/skills --skill git-workflow
```

## Available Skills

### coding-best-practices

General software engineering best practices for writing clean, maintainable, and robust code.

**Use when:**
- Writing new features or modules
- Reviewing or refactoring existing code
- Designing APIs or data structures

**Topics covered:**
- Naming & readability
- Functions & methods (SRP, early returns, pure functions)
- Error handling
- Code organization
- Comments & documentation
- Testing
- Performance & safety
- Dependencies & tooling

### git-workflow

Git branching, commit, and collaboration best practices.

**Use when:**
- Creating branches or writing commit messages
- Opening or reviewing pull requests
- Resolving merge conflicts

**Topics covered:**
- Conventional Commits format
- Branch naming conventions
- Branching strategy (main, develop, feature, hotfix)
- Pull request best practices
- Code review etiquette
- Conflict resolution
- Repository hygiene
